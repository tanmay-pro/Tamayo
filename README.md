# Tamayo

## Running the Application

* Run Mongo daemon:
```
sudo mongod
```
Mongo will be running on port 27017.

* Run Express Backend:
```
cd backend/
npm install
npm start
```

* Run React Frontend:
```
cd frontend
npm install/
npm start
```

Navigate to [http://localhost:3000/](http://localhost:3000/) in your browser.
